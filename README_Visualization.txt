1.0 Functie applicatie 

Deze applicatie is ontwikkeld voor de managers van Rockstars IT.  


1.1 Data 

1.1.1 Data-Dashboard 

Op het dashboard zijn twee verschillende grafieken te zien.  

Een is de barchart, hierin staat alle data van alle questionnaires. 

Onder aan de pagina staat een pie chart. Deze laat de antwoorden zien van de trend vragen. 


1.1.2 Data-Individual 

Op de pagina Individual is per persoon te zien via een barchart wat ze hebben ingevuld per vraag op alle questionnaires. De persoon kies je aan de hand van het emailadres. 

 

1.1.3 Invite 

Het is mogelijk om de data in te zien van de Questionnaire die verstuurd is naar de werknemers van Rockstar. Ook kan de manager de questionnaire versturen naar meerdere werknemers
